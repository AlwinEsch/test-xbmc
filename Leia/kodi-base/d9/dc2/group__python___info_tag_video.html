Detailed Description

Kodi's video info tag class.

Class: InfoTagVideo()

To get video info tag data of currently played source.

Note
Info tag load is only be possible from present player class.

Example:

...
tag = xbmc.Player().getVideoInfoTag()
title = tag.getTitle()
file = tag.getFile()
...
