Detailed Description

Kodi's music info tag class.

Class: InfoTagMusic()

To get music info tag data of currently played source.

Note
Info tag load is only be possible from present player class.

Example:

...
tag = xbmc.Player().getMusicInfoTag()
title = tag.getTitle()
url = tag.getURL()
...
